# README

# WaveShare

## 概要

波情報共有アプリ。<br>日によって変わる波の状況を個人がトピックを立てて投稿。<br>行こうと思っている海の状況をマップ上でピンポイントに確認、共有出来ます。

## バージョン情報

* ruby 2.6.5
* rails 5.2.4
* PostgreSQL


## 機能一覧

* ユーザー登録
* ログイン機能
* コメント機能
* いいね機能
* トピック機能（投稿、一覧、詳細）※Google mapとの連携予定
